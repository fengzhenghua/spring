    • 功能说明及风险提示
    • 功能说明
    • 用于对云销售手机APP验证码发送号码进行维护，因目前登录云销售手机APP需输入验证码，对部分只有一个统一工号的代理商较为不便，通过该功能可将验证码同时发送到设置好的手机号码上，方便代理商使用云销售APP。
    • 风险提示
    • 不及时维护号码造成营业员非法使用可能造成的风险有：
    • 1、营业费用：给非该代理商的营业员登录云销售办理业务，容易造成账实不符的情况。
      2、资金归集：银行卡已绑定工号，如未及时对绑定的号码进行维护，有可能泄露银行卡信息及银行卡内资金。
      3、号码开户：有可能导致用户想要的号码资源被开户，或其他违规开户的情况。
      4、手机终端：沃创富局别的手机丢失、或被套机的现象。
      或其他所有未被考虑在内的风险情况。
      • 云销售只为代理商提供号码绑定的便利，因不妥善使用所造成的一切不良后果均由代理商自行负责，与广西联通无关。
